# Glossary

- **Part:** durable item in the conversation stream.
- **Deck:** collection where running agent work persists independently of open tabs.
- **Engineering-details dial:** vocabulary and density preference, never an entitlement tier.
- **Safe copy / worktree:** audience-specific terms for the same product object.
- **Ship it / Commit & Push:** audience-specific labels for the same action.
- **Steer:** force-send into running work with Command-Enter.
- **Scope:** where a command or new agent runs; selected with Tab in the unified command menu.
