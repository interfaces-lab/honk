# Surface router

| Surface | Load |
| --- | --- |
| Thread, assistant output, composer, queue, `/` or `@` menus | `surfaces-conversation-composer.md` |
| Desktop window, shell, tabs, Home, command menu, workbench | `surfaces-desktop-shell-workbench.md` |
| React Native / mobile client | `surfaces-mobile.md` |

If a user-visible surface is not covered, inspect its source and record the missing decision in
`coverage-gaps.md`; do not generalize a neighboring surface's rule.
