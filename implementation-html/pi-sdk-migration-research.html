Thesis

The right architecture is not "replace Multi with Pi." The right architecture is "make Pi AgentHarness Multi's core-agent system, delete Multi's provider runtime layer, and keep Multi's durable desktop orchestration contract."

Pi already gives Multi the hard provider work: unified model/provider APIs, direct SDK embedding, sessions, tree-shaped JSONL history, queueing, compaction, built-in coding tools, extensions, custom providers, and hooks around provider payloads, tool calls, tool results, context, and session lifecycle. Multi should stop owning separate Codex, Claude, Cursor, ACP, and app-server protocols.

First-principles design: the app has one agent substrate, Pi. Codex is not the base, not a special runtime, and not a package boundary. Codex is one policy-selected Pi provider/model choice, next to Claude and xAI, surfaced through account setup and Agent Mode policy.

Multi should keep what Pi does not try to be: a desktop control plane with persistent projects, timeline/search projections, GUI request panels, live workbench panels, git state, terminals, file trees, queued sends, subagent run panels, settings, and a renderer/runtime-host boundary. Multi should not keep its own conversation tree model; Pi's local JSONL session tree is the source of truth.

Decision

  1. Install Pi as an in-process package dependency and center execution on `AgentHarness`.
  2. Introduce a Multi-owned `ThreadAgentRuntime` service that emits `AgentRuntimeEvent` directly.
  3. Make Pi the model/session/tool authority from the target architecture, with Multi owning product state, UI projections, user-input requests, and persistence.
  4. Expose only the curated login surface: Claude API key, Codex OAuth, Codex API key, and xAI API key.
  5. Make preferences and GUI state the primary rewrite surface: no Pi TUI imports, no provider-instance settings, no generic model picker.
  6. Delete `effect-acp`, `effect-codex-app-server`, old Codex/Claude/Cursor runtime code, provider-specific status probes, and provider registry imports.

Build Multi on Pi, not on Codex. Pi is the base runtime: authentication, model selection, streamed turns, tool calls, cancellation, GUI user input, session persistence, tree recovery, extensions, and built-in coding tools all enter through Pi. The previous question was useful as a capability test: Pi can cover a barebone Codex provider. The first-principles answer is stronger: the replacement target is not Codex. First-Principles Boundary

This does not mean Multi keeps an old runtime-shaped shell and points it at Pi. It means the old runtime shell disappears. The new runtime starts a Pi session with a product-selected provider/model pair, subscribes to Pi events, projects them into `AgentRuntimeEvent`, and stores only desktop read models. The only websocket left in the normal path is Pi's optional provider-side transport to ChatGPT's Codex backend; Multi does not need a websocket server for the local desktop app.

// packages/runtime/src/agent/start-thread.ts
import { getModels } from "@earendil-works/pi-ai";
import {
  AuthStorage,
  createAgentSession,
  ModelRegistry,
  SessionManager,
  SettingsManager,
} from "@earendil-works/pi-coding-agent";

import type { AgentPreferences, ThreadId } from "@multi/contracts";
import { projectPiSessionEvent } from "../projection/projectPiSessionEvent";
import { resolveAgentModePolicy } from "./mode-policy";

export async function startAgentThread(input: {
  readonly threadId: ThreadId;
  readonly cwd: string;
  readonly agentDir: string;
  readonly preferences: AgentPreferences;
  readonly publish: (event: AgentRuntimeEvent) => void;
}): Promise<PiThreadHandle> {
  const authStorage = AuthStorage.create(`${input.agentDir}/auth.json`);
  const modelRegistry = ModelRegistry.create(authStorage, `${input.agentDir}/models.json`);
  const mode = resolveAgentModePolicy({
    agentMode: input.preferences.modelPolicy.defaultAgentMode,
    interactionMode: input.preferences.modelPolicy.defaultInteractionMode,
    role: "main",
    auth: modelRegistry,
  });

  const model = getModels(mode.selection.providerId).find((candidate) =>
    candidate.id === mode.selection.modelId,
  );
  if (!model) {
    throw new Error(`Pi model unavailable: ${mode.selection.providerId}/${mode.selection.modelId}`);
  }

  const { session } = await createAgentSession({
    cwd: input.cwd,
    agentDir: input.agentDir,
    authStorage,
    modelRegistry,
    model,
    thinkingLevel: mode.selection.effectiveThinkingLevel,
    tools: mode.tools.activeToolNames,
    customTools: mode.tools.customTools,
    settingsManager: SettingsManager.create(input.cwd, input.agentDir),
    sessionManager: SessionManager.create(input.cwd, `${input.agentDir}/sessions`),
  });

  const unsubscribe = session.subscribe((event) => {
    input.publish(projectPiSessionEvent({ threadId: input.threadId, event, session }));
  });

  return {
    threadId: input.threadId,
    session,
    async send(prompt, delivery, images) {
      if (delivery === "steer") {
        await session.steer(prompt, images);
        return;
      }
      if (delivery === "follow-up") {
        await session.followUp(prompt, images);
        return;
      }
      await session.prompt(prompt, images ? { images } : undefined);
    },
    async abort() {
      await session.abort();
    },
    getTree() {
      return session.sessionManager.getTree();
    },
    dispose() {
      unsubscribe();
      session.dispose();
    },
  };
}

What Disappears

  • The provider runtime registry as the main runtime router.
  • Separate Codex, Claude, Cursor, ACP, and app-server runtime implementations.
  • Provider-specific session processes and native protocol raw event normalizers.
  • RPC mode as a dependency boundary. Pi RPC can remain useful for external clients, but Multi should not use it internally.
  • The idea that `codex`, `claudeAgent`, and `cursor` are separate runtime drivers. They become Pi model/provider options.

  • Multi's desktop app, runtime host, project model, timeline/search projections, settings UI, git/files/terminal workbench, auth/session control plane, and update flow.
  • Effect as the runtime-host foundation. Pi remains a package dependency wrapped in Effect services at process boundaries.
  • Multi's durable read models and UI-specific projections. Pi owns agent execution state; Multi owns product state.

import type { SessionTreeNode } from "@earendil-works/pi-coding-agent";

type VisiblePiTreeNode = {
  readonly node: SessionTreeNode;
  readonly depth: number;
};

function visiblePiTreeNodes(input: {
  readonly roots: readonly SessionTreeNode[];
  readonly expandedEntryIds: ReadonlySet<string>;
}): readonly VisiblePiTreeNode[] {
  const visible: VisiblePiTreeNode[] = [];

  const visit = (node: SessionTreeNode, depth: number): void => {
    visible.push({ node, depth });
    if (!input.expandedEntryIds.has(node.entry.id)) return;
    for (const child of node.children) {
      visit(child, depth + 1);
    }
  };

  for (const root of input.roots) {
    visit(root, 0);
  }

  return visible;
}

// packages/desktop/src/preload-runtime.ts
import { contextBridge, ipcRenderer } from "electron";
import type { MultiRuntimeApi, RuntimeStream } from "@multi/contracts/runtime-api";

function stream<TEvent>(channel: string, payload?: unknown): RuntimeStream<TEvent> {
  return (listener) => {
    const subscriptionId = crypto.randomUUID();
    const eventChannel = `${channel}:event:${subscriptionId}`;
    const onEvent = (_event: Electron.IpcRendererEvent, value: TEvent) => listener(value);

    ipcRenderer.on(eventChannel, onEvent);
    void ipcRenderer.invoke(`${channel}:subscribe`, { subscriptionId, payload });

    return () => {
      ipcRenderer.off(eventChannel, onEvent);
      void ipcRenderer.invoke(`${channel}:unsubscribe`, { subscriptionId });
    };
  };
}

const runtime = {
  agent: {
    send: (input) => ipcRenderer.invoke("agent.send", input),
    abort: (input) => ipcRenderer.invoke("agent.abort", input),
    getTree: (input) => ipcRenderer.invoke("agent.getTree", input),
    subscribeShell: stream("agent.subscribeShell"),
    subscribeThread: (input, listener) => stream("agent.subscribeThread", input)(listener),
  },
  preferences: {
    getAgentPreferences: () => ipcRenderer.invoke("preferences.getPi"),
    setAgentPreferences: (preferences) => ipcRenderer.invoke("preferences.setPi", preferences),
  },
} satisfies Partial<MultiRuntimeApi>;

contextBridge.exposeInMainWorld("multiRuntime", runtime);

import { Effect, Fiber, Schema, Stream } from "effect";

export type IpcStreamSender = {
  readonly send: (channel: string, value: unknown) => void;
};

export type IpcStreamEvent = {
  readonly sender: IpcStreamSender;
};

export type IpcStreamMain = {
  readonly handle: (
    channel: string,
    listener: (event: IpcStreamEvent, raw: unknown) => Promise<unknown>,
  ) => void;
  readonly removeHandler: (channel: string) => void;
};

const SubscriptionPayload = Schema.Struct({
  subscriptionId: Schema.String,
  payload: Schema.optionalKey(Schema.Unknown),
});

export type IpcStreamRegistration<Payload, EncodedPayload, Event, EncodedEvent, E, R> = {
  readonly channel: string;
  readonly payload: Schema.Codec<Payload, EncodedPayload>;
  readonly event: Schema.Codec<Event, EncodedEvent>;
  readonly stream: (payload: Payload) => Effect.Effect<Stream.Stream<Event, E>, E, R>;
};

export function makeIpcStream<Payload, EncodedPayload, Event, EncodedEvent, E, R>(
  registration: IpcStreamRegistration<Payload, EncodedPayload, Event, EncodedEvent, E, R>,
) {
  return registration;
}

export function installIpcStream<Payload, EncodedPayload, Event, EncodedEvent, E, R>(input: {
  readonly ipcMain: IpcStreamMain;
  readonly registration: IpcStreamRegistration<Payload, EncodedPayload, Event, EncodedEvent, E, R>;
  readonly runPromise: <A>(effect: Effect.Effect<A, E, R>) => Promise<A>;
}): void {
  const subscribeChannel = `${input.registration.channel}:subscribe`;
  const unsubscribeChannel = `${input.registration.channel}:unsubscribe`;
  const fibers = new Map<string, Fiber.RuntimeFiber<void, E>>();
  const decodeSubscription = Schema.decodeUnknownSync(SubscriptionPayload);
  const decodePayload = Schema.decodeUnknownSync(input.registration.payload);
  const encodeEvent = Schema.encodeUnknownSync(input.registration.event);

  input.ipcMain.removeHandler(subscribeChannel);
  input.ipcMain.handle(subscribeChannel, async (event, raw) => {
    const subscription = decodeSubscription(raw);
    const payload = decodePayload(subscription.payload);
    const eventChannel = `${input.registration.channel}:event:${subscription.subscriptionId}`;
    const stream = await input.runPromise(input.registration.stream(payload));
    const fiber = await input.runPromise(
      Stream.runForEach(stream, (value) =>
        Effect.sync(() => event.sender.send(eventChannel, encodeEvent(value))),
      ).pipe(Effect.fork),
    );
    fibers.set(subscription.subscriptionId, fiber);
  });

  input.ipcMain.removeHandler(unsubscribeChannel);
  input.ipcMain.handle(unsubscribeChannel, async (_event, raw) => {
    const subscription = decodeSubscription(raw);
    const fiber = fibers.get(subscription.subscriptionId);
    if (!fiber) return;
    fibers.delete(subscription.subscriptionId);
    await input.runPromise(Effect.interruptFiber(fiber));
  });
}

function isConnected(connection: AccountConnection): boolean {
  switch (connection.type) {
    case "configured":
      return true;
    case "missing":
    case "error":
      return false;
    default: {
      const _exhaustive: never = connection;
      return _exhaustive;
    }
  }
}

function candidateIsAvailable(input: {
  readonly candidate: ModelCandidate;
  readonly accounts: AccountConnections;
}): boolean {
  switch (input.candidate.requires) {
    case "anthropic-api-key":
      return isConnected(input.accounts.anthropic);
    case "codex-oauth":
      return isConnected(input.accounts.codexOAuth);
    case "codex-api-key":
      return isConnected(input.accounts.codexApiKey);
    case "xai-api-key":
      return isConnected(input.accounts.xai);
    default: {
      const _exhaustive: never = input.candidate.requires;
      return _exhaustive;
    }
  }
}

function uniqueRequirements(candidates: readonly ModelCandidate[]): readonly AuthRequirement[] {
  const requirements = new Set<AuthRequirement>();
  for (const candidate of candidates) {
    requirements.add(candidate.requires);
  }
  return [...requirements];
}

function findModel(candidate: ModelCandidate): boolean {
  return getModels(candidate.providerId).some((model) => model.id === candidate.modelId);
}

function requestedThinking(input: {
  readonly agentMode: AgentMode;
  readonly interactionMode: InteractionMode;
  readonly role: AgentRole;
}): ModelThinkingLevel {
  const roleOverride = ROLE_OVERRIDES[input.role].requestedThinkingLevel;
  if (roleOverride) return roleOverride;

  if (input.agentMode === "deep" && input.interactionMode === "plan") return "high";
  return AGENT_MODE_DEFINITIONS[input.agentMode].requestedThinkingLevel;
}

function costPosture(input: {
  readonly role: AgentRole;
  readonly candidate: ModelCandidate;
}): CostPosture {
  const roleCost = ROLE_OVERRIDES[input.role].costPosture;
  return roleCost ?? input.candidate.costPosture;
}

export function resolveAgentMode(input: ResolveModeInput): ModeResolution {
  const agentMode = AGENT_MODE_DEFINITIONS[input.agentMode];
  const interactionMode = INTERACTION_MODE_DEFINITIONS[input.interactionMode];
  const roleOverride = ROLE_OVERRIDES[input.role];
  const candidates = roleOverride.candidates ?? agentMode.candidates;
  const selectedCandidate = candidates.find((candidate) =>
    candidateIsAvailable({ candidate, accounts: input.accounts }),
  );

  if (!selectedCandidate) {
    return { type: "needs-auth", requirements: uniqueRequirements(candidates) };
  }

  if (!findModel(selectedCandidate)) {
    return {
      type: "unsupported-model",
      providerId: selectedCandidate.providerId,
      modelId: selectedCandidate.modelId,
    };
  }

  const piModel = getModels(selectedCandidate.providerId).find((model) =>
    model.id === selectedCandidate.modelId,
  );
  if (!piModel) {
    return {
      type: "unsupported-model",
      providerId: selectedCandidate.providerId,
      modelId: selectedCandidate.modelId,
    };
  }

  const requestedThinkingLevel = requestedThinking(input);

  return {
    type: "resolved",
    mode: {
      agentMode: input.agentMode,
      interactionMode: input.interactionMode,
      role: input.role,
      promptPreset: roleOverride.promptPreset ?? interactionMode.promptPreset,
      toolPosture: roleOverride.toolPosture ?? interactionMode.toolPosture,
      queueBehavior: roleOverride.queueBehavior ?? interactionMode.queueBehavior,
      selection: {
        providerId: selectedCandidate.providerId,
        modelId: selectedCandidate.modelId,
        requestedThinkingLevel,
        effectiveThinkingLevel: clampThinkingLevel(piModel, requestedThinkingLevel),
        costPosture: costPosture({
          role: input.role,
          candidate: selectedCandidate,
        }),
      },
    },
  };
}

import type { AccountConnection, AccountConnections, ProviderId } from "@multi/contracts/agent-preferences";

};

type SetApiKeyInput = {
  readonly providerId: PiApiKeyProviderId;
  readonly key: string;
};

export type AuthGateway = {
  readonly authStorage: AuthStorage;
  readonly snapshot: () => AccountConnections;
  readonly setApiKey: (input: SetApiKeyInput) => void;
  readonly remove: (providerId: ProviderId) => void;
  readonly loginCodexOAuth: (callbacks: OAuthLoginCallbacks) => Promise<void>;
};

function connectionFromStatus(status: AuthStatus): AccountConnection {
  if (status.source) {
    return {
      type: "configured",
      source: status.source,
      ...(status.label ? { label: status.label } : {}),
    };
  }

  if (status.configured) {
    return { type: "configured", source: "stored" };
  }

  return { type: "missing" };
}

export function createAuthGateway(input: CreateAuthGatewayInput): AuthGateway {
  const authStorage = AuthStorage.create(input.authPath);

  function snapshot(): AccountConnections {
    const errors = authStorage.drainErrors();
    const firstError = errors[0];

    if (firstError) {
      const errorConnection = {
        type: "error",
        message: firstError.message,
      } satisfies AccountConnection;

      return {
        anthropic: errorConnection,
        codexOAuth: errorConnection,
        codexApiKey: errorConnection,
        xai: errorConnection,
      };
    }

    return {
      anthropic: connectionFromStatus(authStorage.getAuthStatus("anthropic")),
      codexOAuth: connectionFromStatus(authStorage.getAuthStatus("openai-codex")),
      codexApiKey: connectionFromStatus(authStorage.getAuthStatus("openai")),
      xai: connectionFromStatus(authStorage.getAuthStatus("xai")),
    };
  }

  function setApiKey(input: SetApiKeyInput): void {
    authStorage.set(input.providerId, { type: "api_key", key: input.key });
  }

  function remove(providerId: ProviderId): void {
    authStorage.remove(providerId);
  }

  async function loginCodexOAuth(callbacks: OAuthLoginCallbacks): Promise<void> {
    await authStorage.login("openai-codex", callbacks);
  }

  return { authStorage, snapshot, setApiKey, remove, loginCodexOAuth };
}

This is the default runtime design: the desktop app uses Pi's file-backed `AuthStorage.create(authPath)` under Multi userdata because Pi's storage backend has synchronous writes for API keys, asynchronous writes for OAuth refresh, locked refresh behavior, and 0600 file permissions. If we later replace the backend with OS keychain storage, it must satisfy the same sync and async semantics without changing the GUI contract.

        pi,
        threadId: input.threadId,
        events: input.events,
        preferences: input.preferences,
      }),

    cwd: input.cwd,
    agentDir: input.agentDir,

    systemPrompt: DesktopSystemPrompt.build({ preferences: input.preferences, mode: modeResolution.mode }),
  });

    try: () => resourceLoader.reload(),
    catch: (cause) => new AgentRuntimeError({ message: "Could not reload Pi resources.", cause }),
  });

    Effect.tryPromise({
      try: async () => {
        const { session } = await createAgentSession({
          cwd: input.cwd,
          agentDir: input.agentDir,

          thinkingLevel: modeResolution.mode.selection.effectiveThinkingLevel,

          input.events.emit(AgentRuntimeProjector.fromPi({ threadId: input.threadId, event }));
        });

        return {
          session,
          respondToExtensionUi: extensionUi.respond,
          dispose: () => {
            unsubscribe();
            session.dispose();
          },
        };
      },
      catch: (cause) => new AgentRuntimeError({ message: "Could not create Pi session.", cause }),
    }),
    (handle) => Effect.sync(() => handle.dispose()),
  );

  return handle;
});

export function createDesktopExtensionUi(events: AgentRuntimeEventSink): DesktopExtensionUi {
  const pending = new Map<ExtensionUiRequestId, PendingUiRequest>();

  function request<T>(input: {
    readonly options: ExtensionUIDialogOptions | undefined;
    readonly fallback: T;
    readonly makeRequest: (requestId: ExtensionUiRequestId) => ExtensionUiRequest;
    readonly readResponse: (response: ExtensionUiResponse) => T;
  }): Promise<T> {
    if (input.options?.signal?.aborted) return Promise.resolve(input.fallback);

    const requestId = ExtensionUiRequestId.make(randomUUID());

    return new Promise<T>((resolve) => {
      let timeout: ReturnType<typeof setTimeout> | undefined;

      const cleanup = () => {
        if (timeout) clearTimeout(timeout);
        input.options?.signal?.removeEventListener("abort", onAbort);
        pending.delete(requestId);
      };

      const finish = (value: T) => {
        cleanup();
        resolve(value);
      };

      const onAbort = () => finish(input.fallback);
      input.options?.signal?.addEventListener("abort", onAbort, { once: true });

      if (input.options?.timeout) {
        timeout = setTimeout(() => finish(input.fallback), input.options.timeout);
      }

      pending.set(requestId, {
        requestId,
        resolve: (response) => finish(input.readResponse(response)),
      });
      events.emit({ type: "extension_ui.requested", request: input.makeRequest(requestId) });
    });
  }

  function respond(response: ExtensionUiResponse): void {
    const pendingRequest = pending.get(response.requestId);
    if (!pendingRequest) return;
    pendingRequest.resolve(response);
  }

  return {
    respond,
    context: {
      select: (title, options, dialogOptions) =>
        request({
          options: dialogOptions,
          fallback: undefined,
          makeRequest: (requestId) => ({
            type: "select",
            requestId,
            title,
            options,
            ...(dialogOptions?.timeout ? { timeoutMs: dialogOptions.timeout } : {}),
          }),
          readResponse: (response) => (response.type === "value" ? response.value : undefined),
        }),
      confirm: (title, message, dialogOptions) =>
        request({
          options: dialogOptions,
          fallback: false,
          makeRequest: (requestId) => ({
            type: "confirm",
            requestId,
            title,
            message,
            ...(dialogOptions?.timeout ? { timeoutMs: dialogOptions.timeout } : {}),
          }),
          readResponse: (response) => (response.type === "confirmed" ? response.confirmed : false),
        }),
      input: (title, placeholder, dialogOptions) =>
        request({
          options: dialogOptions,
          fallback: undefined,
          makeRequest: (requestId) => ({
            type: "input",
            requestId,
            title,
            ...(placeholder ? { placeholder } : {}),
            ...(dialogOptions?.timeout ? { timeoutMs: dialogOptions.timeout } : {}),
          }),
          readResponse: (response) => (response.type === "value" ? response.value : undefined),
        }),
      editor: (title, prefill) =>
        request({
          options: undefined,
          fallback: undefined,
          makeRequest: (requestId) => ({ type: "editor", requestId, title, ...(prefill ? { prefill } : {}) }),
          readResponse: (response) => (response.type === "value" ? response.value : undefined),
        }),
      notify: (message, type) => events.emit({ type: "extension_ui.notify", message, notifyType: type }),
      setStatus: (key, text) => events.emit({ type: "extension_ui.status", key, text }),
      setWidget: (key, lines) => events.emit({ type: "extension_ui.widget", key, lines }),
      setTitle: (title) => events.emit({ type: "extension_ui.title", title }),
      setEditorText: (text) => events.emit({ type: "extension_ui.set_editor_text", text }),
      onTerminalInput: () => () => {},

              content: [{ type: "text", text: "User cancelled the question." }],
              details: { type: "cancelled", questions, reason: "cancelled" },
            };

        const option = question.options[optionIndex];

          return {
            content: [{ type: "text", text: "User cancelled the question." }],
            details: { type: "cancelled", questions, reason: "invalid selection" },
          };
        }

          id: question.id,
          value: option.value,
          label: option.label,
          wasCustom: false,
          index: optionIndex + 1,
        });
      }

        content: [{ type: "text", text: answerText(answers) }],
        details: { type: "answered", questions, answers },
      };
    },
  });

function truncateForParent(text: string): string {
  const bytes = Buffer.byteLength(text, "utf8");
  if (bytes <= PER_TASK_OUTPUT_CAP) return text;

  let truncated = text.slice(0, PER_TASK_OUTPUT_CAP);
  while (Buffer.byteLength(truncated, "utf8") > PER_TASK_OUTPUT_CAP) {
    truncated = truncated.slice(0, -1);
  }
  return `${truncated}\n\n[Output truncated: ${bytes - Buffer.byteLength(truncated, "utf8")} bytes omitted. Full output is visible in the child run.]`;
}

function requestedAgentNames(params: SubagentParams): readonly string[] {

function oneModeSelected(params: SubagentParams): boolean {
  const hasChain = (params.chain?.length ?? 0) > 0;
  const hasTasks = (params.tasks?.length ?? 0) > 0;

async function mapWithConcurrencyLimit<TIn, TOut>(
  items: readonly TIn[],
  concurrency: number,
  run: (item: TIn, index: number) => Promise<TOut>,
): Promise<readonly TOut[]> {
  const results: TOut[] = new Array(items.length);
  let nextIndex = 0;
  const workers = new Array(Math.min(concurrency, items.length)).fill(null).map(async () => {
    while (nextIndex < items.length) {
      const index = nextIndex++;
      results[index] = await run(items[index], index);
    }
  });
  await Promise.all(workers);
  return results;
}

        projectAgentsDir: discovery.projectAgentsDir,
        results,
      });

      if (!oneModeSelected(params)) {
        const available = discovery.agents.map((agent) => `${agent.name} (${agent.source})`).join(", ") || "none";
        return {
          content: [{ type: "text", text: `Invalid subagent parameters. Available agents: ${available}` }],
          details: makeDetails("single")([]),
          isError: true,
        };
      }

      if ((agentScope === "project" || agentScope === "both") && confirmProjectAgents && ctx.hasUI) {
        const projectAgents = requestedAgentNames(params)
          .map((name) => discovery.agents.find((agent) => agent.name === name))
          .filter((agent): agent is SubagentDefinition => agent?.source === "project");

        if (projectAgents.length > 0) {
          const names = projectAgents.map((agent) => agent.name).join(", ");
          const approved = await ctx.ui.confirm(
            "Run project-local agents?",
            `Agents: ${names}\nSource: ${discovery.projectAgentsDir ?? "(unknown)"}\n\nProject agents are repo-controlled. Only continue for trusted repositories.`,
            { signal },
          );
          if (!approved) {
            return {
              content: [{ type: "text", text: "Canceled: project-local agents not approved." }],
              details: makeDetails(params.chain?.length ? "chain" : params.tasks?.length ? "parallel" : "single")([]),
            };
          }
        }
      }

      const runOne = async (
        agentName: string,
        task: string,
        cwd: string | undefined,
        mode: SubagentToolDetails["mode"],
      ): Promise<SubagentRunResult> => {
        const agent = discovery.agents.find((candidate) => candidate.name === agentName);
        if (!agent) {
          const available = discovery.agents.map((candidate) => `"${candidate.name}"`).join(", ") || "none";
          return {
            childThreadId: input.threadId,
            agent: agentName,
            agentSource: "unknown",
            task,
            status: "failed",
            finalText: `Unknown agent: "${agentName}". Available agents: ${available}.`,
          };
        }

        return input.host.run({
          parentThreadId: input.threadId,
          cwd: cwd ?? ctx.cwd,
          agent,
          task,
          signal,
          preferences: input.preferences,
          onUpdate: (result) => {
            input.events.emit({ type: "subagent_run.updated", parentThreadId: input.threadId, result });
            onUpdate?.({
              content: [{ type: "text", text: result.finalText || "(running...)" }],
              details: makeDetails(mode)([result]),
            });
          },
        });
      };

      if (params.chain?.length) {
        const results: SubagentRunResult[] = [];
        let previous = "";
        for (const step of params.chain) {
          const task = step.task.replace(/\{previous\}/g, previous);
          const result = await runOne(step.agent, task, step.cwd, "chain");
          results.push(result);
          if (result.status !== "succeeded") {
            return {
              content: [{ type: "text", text: `Chain stopped at ${step.agent}: ${result.finalText}` }],
              details: makeDetails("chain")(results),
              isError: true,
            };
          }
          previous = result.finalText;
        }
        return {
          content: [{ type: "text", text: results.at(-1)?.finalText ?? "(no output)" }],
          details: makeDetails("chain")(results),
        };
      }

      if (params.tasks?.length) {
        if (params.tasks.length > MAX_PARALLEL_TASKS) {
          return {
            content: [{ type: "text", text: `Too many parallel subagent runs. Max is ${MAX_PARALLEL_TASKS}.` }],
            details: makeDetails("parallel")([]),
            isError: true,
          };
        }

        const results = await mapWithConcurrencyLimit(params.tasks, MAX_CONCURRENCY, (task) =>
          runOne(task.agent, task.task, task.cwd, "parallel"),
        );
        const successCount = results.filter((result) => result.status === "succeeded").length;
        const summaries = results.map((result) =>
          `### [${result.agent}] ${result.status}\n\n${truncateForParent(result.finalText)}`,
        );
        return {
          content: [{ type: "text", text: `Parallel: ${successCount}/${results.length} succeeded\n\n${summaries.join("\n\n---\n\n")}` }],
          details: makeDetails("parallel")(results),
        };
      }

      const result = await runOne(params.agent!, params.task!, params.cwd, "single");
      return {
        content: [{ type: "text", text: result.finalText || "(no output)" }],
        details: makeDetails("single")([result]),
        ...(result.status === "succeeded" ? {} : { isError: true }),
      };
    },
  });
}

function finalTextFromSession(session: { readonly messages: readonly { readonly role: string; readonly content: unknown }[] }): string {
  for (let index = session.messages.length - 1; index >= 0; index--) {
    const message = session.messages[index];
    if (message.role !== "assistant" || !Array.isArray(message.content)) continue;
    for (const part of message.content) {
      if (typeof part === "object" && part !== null && "type" in part && part.type === "text" && "text" in part) {
        return typeof part.text === "string" ? part.text : "";
      }
    }
  }
  return "";
}

export function createEmbeddedSubagentRunHost(input: CreateEmbeddedSubagentRunHostInput): SubagentRunHost {
  return {
    async run(runInput): Promise<SubagentRunResult> {
      const childThreadId = ThreadId.make(`${runInput.parentThreadId}:subagent-run:${crypto.randomUUID()}`);
      const mode = AgentModePolicy.resolve({
        agentMode: runInput.preferences.modelPolicy.defaultAgentMode,
        interactionMode: "agent",
        role: "subagent",
        accounts: runInput.preferences.accounts,
        requestedModelId: runInput.agent.model,
        requestedTools: runInput.agent.tools,
      });
      if (mode.type !== "resolved") {
        return {
          childThreadId,
          agent: runInput.agent.name,
          agentSource: runInput.agent.source,
          task: runInput.task,
          status: "failed",
          finalText: `Could not resolve subagent model policy: ${mode.type}`,
        };
      }

      const model = getModels(mode.mode.selection.providerId).find((candidate) =>
        candidate.id === mode.mode.selection.modelId,
      );
      if (!model) {
        return {
          childThreadId,
          agent: runInput.agent.name,
          agentSource: runInput.agent.source,
          task: runInput.task,
          status: "failed",
          finalText: `Pi model unavailable: ${mode.mode.selection.providerId}/${mode.mode.selection.modelId}`,
        };
      }

      const extensionUi = createDesktopExtensionUi({
        emit: (event) => runInput.onUpdate({
          childThreadId,
          agent: runInput.agent.name,
          agentSource: runInput.agent.source,
          task: runInput.task,
          status: "succeeded",
          finalText: event.type,
        }),
      });

      const resourceLoader = new DefaultResourceLoader({
        cwd: runInput.cwd,
        agentDir: input.agentDir,
        settingsManager: input.settingsManager,
        systemPrompt: runInput.agent.systemPrompt,
        noPromptTemplates: true,
      });
      await resourceLoader.reload();

      const { session } = await createAgentSession({
        cwd: runInput.cwd,
        agentDir: input.agentDir,
        authStorage: input.authStorage,
        modelRegistry: input.modelRegistry,
        settingsManager: input.settingsManager,
        resourceLoader,
        sessionManager: SessionManager.inMemory(runInput.cwd),
        model,
        thinkingLevel: mode.mode.selection.effectiveThinkingLevel,
        tools: mode.mode.tools.activeToolNames,
      });

      try {
        await session.bindExtensions({ uiContext: extensionUi.context });
        const unsubscribe = session.subscribe(() => {
          runInput.onUpdate({
            childThreadId,
            agent: runInput.agent.name,
            agentSource: runInput.agent.source,
            task: runInput.task,
            status: "succeeded",
            finalText: finalTextFromSession(session),
            modelId: model.id,
          });
        });

        try {
          await session.prompt(`Task: ${runInput.task}`, { signal: runInput.signal });
        } finally {
          unsubscribe();
        }

        return {
          childThreadId,
          agent: runInput.agent.name,
          agentSource: runInput.agent.source,
          task: runInput.task,
          status: runInput.signal?.aborted ? "aborted" : "succeeded",
          finalText: finalTextFromSession(session) || "(no output)",
          modelId: model.id,
        };
      } catch (cause) {
        return {
          childThreadId,
          agent: runInput.agent.name,
          agentSource: runInput.agent.source,
          task: runInput.task,
          status: runInput.signal?.aborted ? "aborted" : "failed",
          finalText: cause instanceof Error ? cause.message : "Subagent run failed.",
          modelId: model.id,
        };
      } finally {
        session.dispose();
      }
    },
  };
}

Rewrite Plan

  1. Define the desktop Agent UI first. Create the `AgentPreferences` contract, new preferences page, account rows, interaction-mode picker, pending request panel, and Pi `/tree` view so core logic has a product surface without TUI.
  2. Define the runtime boundary as IPC first. Replace websocket/default-server assumptions with `DesktopRuntimeHost`, preload `MultiRuntimeApi`, and typed IPC request/stream channels.
  3. Cut the old runtime packages. Remove `effect-acp`, `effect-codex-app-server`, native provider runtimes, provider status probes, provider-specific settings, websocket route code, server exposure, and generated protocol dependencies from the package graph.
  4. Delete `packages/client-runtime`. Collapse known-environment and scoped-ref helpers into the app or remove them with single-environment desktop boot.
  5. Add Pi as the core-agent dependency. Install Pi packages and create `ThreadAgentRuntime` around `AgentHarness`, Pi session storage, Pi resource loading, and Pi extensions.
  6. Create the canonical agent identity. Define `agentRuntime`, `authProviderId`, `modelId`, `accountId`, `threadId`, and `piSessionId` as the only runtime identity shape.
  7. Delete Multi's conversation tree model. Use Pi JSONL session persistence, `SessionManager.getTree()`, `navigateTree()`, labels, and branch summarization as the source of truth. Keep filters as renderer-only view predicates.
  8. Emit `AgentRuntimeEvent` directly. Rewrite orchestration ingestion to consume the agent event contract without adding another provider-shaped runtime path.
  9. Map events comprehensively. Cover messages, reasoning, tools, file changes, command output, queue updates, compaction, usage, errors, and user input.
  10. Keep access posture fixed. Delete provider-era permission/runtime-mode settings. Multi's Pi runtime runs all-access; interaction modes only change prompt/tool/UI posture.
  11. Build curated GUI login. Replace Codex/Claude/Cursor settings cards with Claude API key, Codex OAuth, Codex API key, and xAI API key. Use Pi `AuthStorage`, OAuth callbacks, and a Multi-owned Pi auth path.
  12. Force model selection through `AgentModePolicy`. Agent Mode chooses blessed Pi provider/model pairs and effective reasoning level; Interaction Mode chooses prompt, active tools, queue behavior, and UI posture. The user picks auth and intent, not a generic model registry row.
  13. Add GUI ask/plan/subagent extensions. Implement as Pi extension factories backed by Multi UI events and orchestration projections.
  14. Rebuild tests around the target architecture. Prefer package-level `pnpm run typecheck`, targeted `ThreadAgentRuntime` tests, user-input tests, and app browser tests for login, pending panels, timeline, Agent Mode, and Interaction Mode.
